features = ['year', 'month', 'day', 'weekday', 'hour', 'step', 'rating']

# Reducir a 3 dimensiones con PCA

# Crear DataFrame con embeddings

Descripcin de las Variables Temporales Generadas

Variable Tipo Descripcin Fuente de datos originales
temporal_1 Numrica Primera componente principal generada mediante PCA sobre las variables temporales. Captura la direccin con mayor varianza en el comportamiento temporal de los usuarios. year, month, day, weekday, hour, step, rating
temporal_2 Numrica Segunda componente principal del anlisis temporal. Representa patrones adicionales de comportamiento no capturados por temporal_1. Mismas variables que temporal_1
temporal_3 Numrica Tercera componente principal. Complementa las dos anteriores para representar mejor las dinmicas de tiempo de cada interaccin. Mismas variables que temporal_1
step Entera Representa el orden relativo en el que un usuario calific las pelculas. Es til para modelar el historial de comportamiento como secuencia. Derivado de timestamp, agrupado por userId
