# Crear DataFrame con embeddings

🧠 Descripción de las Variables Temporales Generadas¶

Variable Tipo Descripción Fuente de datos originales
temporal_1 Numérica Primera componente principal generada mediante PCA sobre las variables temporales. Captura la dirección con mayor varianza en el comportamiento temporal de los usuarios. year, month, day, weekday, hour, step, rating
temporal_2 Numérica Segunda componente principal del análisis temporal. Representa patrones adicionales de comportamiento no capturados por temporal_1. Mismas variables que temporal_1
temporal_3 Numérica Tercera componente principal. Complementa las dos anteriores para representar mejor las dinámicas de tiempo de cada interacción. Mismas variables que temporal_1
step Entera Representa el orden relativo en el que un usuario calificó las películas. Es útil para modelar el historial de comportamiento como secuencia. Derivado de timestamp, agrupado por userId
